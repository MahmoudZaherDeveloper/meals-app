## Tech Stack
- kotlin. 
- Clean Architecure with (MVVM + StateManagement).
- Coroutines.
- Retrofit2.
- recycler view.
- ConstraintLayout.
- viewmodel and StateFlow.
- view binding.
- Dagger Hilt for DI.
